When creating a project that will contain modules for Mayhem, reference the following:

NuGetPackages:
MayhemCore

Assemblies:
System.Runtime.Serialization

If your modules are configurable and have configuration windows, also reference the following:

NuGetPackages:
MayhemWpf

Assemblies:
PresentationCore
PresentationFramework
System.Xaml
WindowsBase
